## 한 줄 요약

## 핵심 개념 용어

| 용어 | 의미 |
|------|------|
| **천적응애 (predator)** | 해충을 잡아먹는 익충. 사육·출하 대상 (클래스 0) |
| **먹이응애 (prey)** | 천적응애의 먹이가 되는 응애 (클래스 1) |
| **사육박스 (culture box)** | 응애를 배양하는 단위. 모든 측정의 기준 컨텍스트 |
| **밀도 (density)** | mL/L당 천적응애 개체 수. 출하 가능성 판단 지표 |
| **활력도 (vitality)** | 천적응애 이동 속도 기반 0~100점 활력 점수 |
| **증식률 (growth)** | 시간에 따른 밀도·활력도 변화율(증식 속도) |
